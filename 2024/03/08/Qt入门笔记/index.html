QT入门笔记

  • 注意事项：

    界面设计的search使用时把要搜索的内容的前面加上一个“q”

    C++中头文件h与源文件cpp之间的切换快捷键：F4

    在ui界面上面修改控件名字后，要编译程序ctrl+b

    通过qDebug()<<输出调试内容，在应用程序输出查看

    只要是Widget都是属于控件，最底层的Widgets都是窗体

    如果指针在使用的时候，我们将指针变量名称加上，意味着取得指针指向的内存的值。就是说代表取值

    自动在cpp文件生成定义的快捷键：Alt+Enter

    快捷键alt+/可以代码提示

    sqlite3 wsmdlr.db 表示创建一个数据库文件名称为wsmdlr.db
    .tables表示显示当前数据库中所有的表有哪些
    create table userinfo(username,password)创建userinfo数据表
    向数据表userinfo中插入一行记录wsmdlr|123
    insert into userinfo(username,password) values(‘wsmdlr’,123);
    查询userinfo表中的所有数据
    select * from userinfo;
    更新数据库中所有的数据记录用update
    更新所有行的password字段的值
    update userinfo set password=123222;

    更新某些指定的记录对应的password，此时需要用where语句进行筛选定位
    update userinfo set password=123 where username=’wsmdlr’;

    如果想要清空某张表，使用delete语句
    delete from userinfo;清空userinfo表中所有的记录

    想要清除指定行的记录，就要结合使用where语句
    delete from userinfo where username=’wsmdlr’;

    通过qt使用sqlite数据库
    引入sql类库
    在pro工程文件中引入sql
    引入方法QT+=sql
    另一种方式就是在原来的基础上拼接，用空格隔开，每一个空格隔开的表示一个类库
    窗体头文件引入QSqlDatabase就可以链接数据库、创建数据库
    db=QSqlDatabase::addDatabase(“QSQLITE”);通过这个来创建需要链接的数据库类型。
    通过setDatabaseName方法创建数据库文件

    窗体引入QSqlQuery就可以执行数据库语句了
    通过QSqlQuery的exec方法来执行sql语句

    qt操作的sqlite如果数据库或表已经存在，那么是不会再次创建，而是打开

    qlable
    qbutton
    qlineedit
    表格网格控件
    qtableview控件显示数据库中的内容为表格行
    修改窗体的标题的属性：windowtitle
    如果实在写的时候找不到，要学会使用帮助文档参考。
    实现数据库对象的初始化，要使用QSqlDatabase静态方法
    db=QSqlDatabase::addDatabase(“QSQLITE”);

    qtableview控件通过QSqlQueryModel来渲染数据
    1.通过setModel的形式加入QSqlQueryModel对象
    2.通过QSqlQueryModel来从数据库中查询出需要的数据
    3.如果要修改显示的表头方法：
    法一：通过数据库查询字段取别名的方式 ，使用as关键词
    select username as 用户名,password as 密码 from userinfo
    法二：通过QSqlQueryModel的设置表头的方法来实现
    通过模型来设置表头字段的别名，使用setHeaderData方法

    tableview表格内横向列扩展，使用属性horizontalheaderstretchlastsection

    获取tableview中当前点击行的数据，中间的0表示第0列。
    QString username=currentIndexModel.siblingAtColumn(0).data().toString();

  • 概念：

    继承：一个类在继承一个父类后，会把父类的方法都继承下来

    qt中槽：代表方法，我们需要通过信号到槽方法中

    信号：理解为事件

    ::可以访问C++类最终的static定义的方法

    →访问类对象中的某个属性或方法

    private：私有化，只有类内部才能使用

    public：公共的，可以把这个属性开放出去使用，这样在面向对象的开发中失去了意义

    protect：保护的，程序类库级别使用

    窗体出现的本质：窗体被死循环

    slot槽：方法（函数）

    signal信号：事件

    *：代表指针

    &：代表取指针指向的地址，此时修改的值是最为彻底。（取址）

    connect方法：信号槽方法

    connect方法的四个参数依次是：

    信号发送者的指针，发送了什么信号，接收者的指针接收者的槽函数，接着是槽方法

    QT4：connect(timer1,SIGNAL(timeout()),this,SLOT(daojishi()));

    QT5：connect(timer1,&QTimer::timeout,this,&text::daojishi);
